Build Your Own AI Tools - It's That Easy

100% open source. Extend with custom tools using our simple decorator pattern.

Dead Simple Decorator

Just add @llm_tool to any Python method. That's it!

from odoo.addons.llm_tool.decorators import llm_tool class SaleOrder(models.Model): _inherit = "sale.order" @llm_tool(read_only_hint=True) def get_pending_orders(self, customer_id: int) -> list: """Get all pending orders for a customer""" orders = self.search([ ('partner_id', '=', customer_id), ('state', 'in', ['draft', 'sent']) ]) return [{ 'id': o.id, 'name': o.name, 'amount_total': o.amount_total } for o in orders]
Auto-registered! Tool appears in AI clients instantly on next Odoo restart.
